# 🚀 LLM Prompt Optimization & Evaluation System

## 🧠 Overview
A full-stack AI system that allows users to compare responses from multiple Large Language Models (LLMs), evaluate their performance, and optimize prompts for better results.

---

## ✨ Features

- 🔍 Compare outputs from multiple AI models (ChatGPT, Claude, Gemini)
- ⭐ Rate responses using a star-based evaluation system
- 🏆 Select best model based on quality, accuracy, or creativity
- ⚡ Real-time prompt optimization
- 📊 Smart auto-evaluation logic
- 💾 Save evaluations locally

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Framer Motion

### Backend
- Python (Flask)
- Flask-CORS

---

## 🧩 System Architecture

Frontend (React) ↔ Backend (Flask API)

- Frontend handles UI, state, and interactions
- Backend simulates LLM responses and processes prompts
